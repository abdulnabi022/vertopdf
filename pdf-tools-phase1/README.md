# RarePDFtool

All-in-one PDF & image toolkit. Fast, private, and free. No files are stored after processing.

## Features
- Merge PDFs
- Compress PDF
- Images to PDF
- PDF to JPG
- JPG to PDF
- PDF to PNG
- PNG to PDF
- Image Converter

## Usage
Run the frontend and backend locally, then open the app in your browser.
